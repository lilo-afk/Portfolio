## ⚙️ Rodando localmente

### Pré-requisitos

```
# Clone o repositório
$ git clone <https://github.com/lilo-afk/Portfolio.git>

# Acesse a pasta do projeto no terminal/cmd
$ cd Portfolio

# Instale as dependências
$ npm i

# Execute a aplicação

$ npm run dev
```
